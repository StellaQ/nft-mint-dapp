# NFT Mint DApp 项目需求规格

## 项目概述
这是一个基于 Web3 的 NFT 铸造 DApp（Decentralized Application），允许用户连接钱包、查看余额、铸造 NFT 并浏览已拥有的 NFT 画廊。

## 核心功能需求

### 1. 钱包连接 (Connect)
- 支持主流 Web3 钱包（如 MetaMask）的连接
- 显示连接状态和当前账户地址
- 处理连接失败和用户拒绝的情况

### 2. 余额显示
- 显示用户当前账户的 ETH 余额
- 实时更新余额信息
- 支持不同网络的余额查询

### 3. NFT 铸造 (Mint)
- 提供 mint() 函数调用接口
- 支持用户输入铸造参数（如数量、价格等）
- 显示铸造状态和交易确认
- 处理交易失败和 gas 费估算

### 4. NFT 画廊 (Gallery)
- 从智能合约读取用户的 tokenURI
- 显示用户拥有的 NFT 列表
- 支持 NFT 图片和元数据的展示
- 提供分页或滚动浏览功能

## 技术要求
- 前端框架：React + TypeScript
- Web3 库：ethers.js 或 web3.js
- 样式：Tailwind CSS 或 Material-UI
- 响应式设计，支持移动端

## 最终目标
1. **代码推送**：将完整代码推送到 GitHub 仓库
2. **在线部署**：项目部署在 GitHub Pages 或类似平台，实现在线访问

## 非功能需求
- 安全性：确保用户资金安全，防止常见 Web3 攻击
- 用户体验：简洁直观的操作流程
- 性能：快速加载和响应
- 兼容性：支持主流浏览器和钱包
